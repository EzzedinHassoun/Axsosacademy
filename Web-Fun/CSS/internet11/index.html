So, what's an Internet?

An Interconnected Network of computers all around the world!

Clients and Servers

Computers connected to the Web are called clients and servers.

  • Clients are devices like your computer or phone, with Web-accessing software.
  • Servers store webpages, sites, or apps. When a client requests data, the server responds.

So What Actually Happens?

  1. The browser goes to the DNS server to find the real address.
  2. The browser sends an HTTP request to the server.
  3. The server replies with a 200 OK message and starts sending the files.
  4. The browser assembles everything into a full webpage.

The Big 3 - HTML, CSS, JavaScript

Files served to the client are always a mix of HTML, CSS, and JavaScript.
